## [1.0.0] - 2024-11-26

### Added
- Initial stable release
- Core ProofClient implementation
- API request recording functionality
- Record retrieval methods
- Export functionality for compliance reports
- Full TypeScript support with type definitions
- Comprehensive examples and documentation
- Integration with Polygon blockchain
- IPFS storage integration
- E2E encryption for sensitive data
- Support for PUBLIC, PRIVATE, and SHARED visibility levels
- Automatic blockchain anchoring
- MiFID II compliance features
- 7-year data retention support

### Security
- Environment-based configuration
- Input validation and sanitization
